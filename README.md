## Conceitos abordados

- Classes e objetos
- Encapsulamento (getters e setters)
- Herança e polimorfismo
- Abstração
- Construtores e sobrecarga de métodos

## Tecnologias

- Java 17+
- IDE: IntelliJ IDEA / VS Code

## Como executar

```bash
# Clone o repositório
git clone https://github.com/devfalcomer/poo-java.git

# Acesse a pasta
cd poo-java

# Compile e execute a classe desejada
javac NomeDaClasse.java
java NomeDaClasse
```

## Objetivo

Este repositório faz parte da minha jornada de transição para 
Desenvolvedor Back-End Java. Os conceitos de POO aqui praticados 
são a base para projetos com Spring Boot e APIs REST.
